# Dinkum Support for Vortex

## Disclaimer
This is the first (game)-extension I wrote for Vortex. So it might have issues. 
Feel free to message me in the Dinkum Discord Server (Channel: #modding_space) and add @Kjay9558.

## BepInEx
BepInEx is needed for every mod provided on Nexusmods/dinkum, make sure to install manually

## TR Tools
TR Tools is needed for some mods, make sure to install it manually if the mod tells you it is dependant on this API.
